# 锂离子电池数据分析

本仓库包含锂离子电池降解实验的分析工具和结果。数据集包括四个锂离子电池（#53-56）在4°C环境温度下经受各种操作条件测试的数据。

## 数据集描述

### 电池测试规格
- **电池编号**: #53, #54, #55, #56（共四个锂离子电池）
- **环境温度**: 4°C
- **操作模式**: 充电、放电和阻抗（EIS）

#### 充电协议
- **模式**: 恒流（CC）1.5A充电至4.2V
- **然后**: 恒压（CV）模式直到充电电流降至20mA

#### 放电协议
- **负载电流**: 固定2A
- **停止电压**:
  - 电池 #53: 2.0V
  - 电池 #54: 2.2V
  - 电池 #55: 2.5V
  - 电池 #56: 2.7V

#### 阻抗测量
- **方法**: 电化学阻抗谱（EIS）
- **频率范围**: 0.1Hz 至 5kHz

### 测试时长
- 实验持续进行直到容量降至1.4 Ahr（30%衰减）
- 注意：某些放电循环显示出非常低的容量（原因尚未完全分析）

## 数据结构

每个 `.mat` 文件包含一个顶层结构数组，其中包含循环数据：

### 循环字段
- `type`: 操作类型（充电、放电或阻抗）
- `ambient_temperature`: 环境温度（°C）
- `time`: 循环开始的日期和时间（MATLAB日期向量格式）
- `data`: 测量数据结构

### 按类型分类的数据字段

#### 充电数据
- `Voltage_measured`: 电池端电压（V）
- `Current_measured`: 电池输出电流（A）
- `Temperature_measured`: 电池温度（°C）
- `Current_charge`: 充电器电流（A）
- `Voltage_charge`: 充电器电压（V）
- `Time`: 时间向量（秒）

#### 放电数据
- `Voltage_measured`: 电池端电压（V）
- `Current_measured`: 电池输出电流（A）
- `Temperature_measured`: 电池温度（°C）
- `Current_load`: 负载电流（A）
- `Voltage_load`: 负载电压（V）
- `Time`: 时间向量（秒）
- `Capacity`: 电池容量（Ahr），放电至2.7V

#### 阻抗数据
- `Sense_current`: 感应支路电流（A）
- `Battery_current`: 电池支路电流（A）
- `Current_ratio`: 电流比
- `Battery_impedance`: 从原始数据计算的电池阻抗（Ω）
- `Rectified_impedance`: 校准和平滑后的阻抗（Ω）
- `Re`: 估算的电解质电阻（Ω）
- `Rct`: 估算的电荷转移电阻（Ω）

## 分析脚本

### 安装依赖

```bash
pip install scipy numpy matplotlib pandas
```

或使用提供的依赖文件：

```bash
pip install -r requirements.txt
```

### 使用方法

运行完整分析：

```bash
python3 analyze_battery_data.py
```

脚本将会：
1. 加载所有电池数据文件
2. 分析数据结构并生成统计信息
3. 为每个电池创建可视化图表
4. 生成所有电池的对比图
5. 将所有结果保存在 `output/` 目录中

## 分析结果

脚本生成的综合分析包括：

### 1. 容量衰减分析
- 跟踪放电循环中的电池容量
- 识别容量衰减模式
- 每个电池的独立图表
- 所有电池的对比图

### 2. 电压-电流曲线
- 充电电压随时间的变化曲线
- 放电电压随时间的变化曲线
- 多个循环叠加对比

### 3. 温度分析
- 充电过程中的温度曲线
- 放电过程中的温度曲线
- 温度范围和变化分析

### 4. 阻抗分析（EIS）
- Nyquist图（实部 vs 虚部阻抗）
- 阻抗随循环的演变
- 电解质和电荷转移电阻

### 5. 汇总统计
对每个电池：
- 按类型分类的总循环次数
- 容量统计（初始、最终、衰减百分比）
- 温度统计（平均值、最小值、最大值、标准差）

## 进一步分析方向

可能的进一步研究方向：

1. **容量恢复**: 为什么电池#55和#56显示容量增加？
2. **低容量循环**: 调查具有非常低容量的放电循环
3. **温度效应**: 分析温度与容量衰减之间的相关性
4. **阻抗增长**: 研究阻抗如何随电池降解而演变
5. **健康状态预测**: 开发模型预测剩余使用寿命
